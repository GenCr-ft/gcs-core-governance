# Readme

This section of the Gencraft Studio Handbook is dedicated to our policies and procedures for the long-term preservation, management, and disposition of studio information and digital assets. It focuses on establishing clear guidelines for archiving and data retention to ensure compliance, operational efficiency, and the safeguarding of valuable studio resources.

Refer to the documents indexed below for detailed information on our archive catalog and the design of our archiving systems.

---

## Index of Contents

### Files

- [Archive Catalog](PRO-STAN-002.archive-catalog.md): The Archive Catalog provides a centralized location for managing and
accessing archived data within Google Cloud Storage. It facilitates efficient retrieval and reporting on
historical data, ensuring compliance and usability.
- [Archiving System Design](PRO-SPEC-002.archiving-system-design.md): This document outlines the design principles for a
robust archiving system utilizing GCS Studio, focusing on data preservation, retention policies, and efficient
retrieval strategies. It details considerations for scalability and integration within a broader data
management framework.
