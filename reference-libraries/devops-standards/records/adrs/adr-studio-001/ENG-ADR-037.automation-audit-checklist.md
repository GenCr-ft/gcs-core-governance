## 1. Objective

This document serves as the master checklist for the audit of all existing automation components (CI/CD
pipelines, scripts, GitHub Actions, etc.) within the `GenCr-ft` GitHub organization. The audit aims to
identify all elements affected by the repository renaming and restructuring mandated by `adr-studio-001.
md`. This checklist will then guide the planning and execution of necessary updates to these automations.

## 2. Scope of Audit

The audit covers:

- All GitHub Actions workflows located in `.github/workflows/` directories across all repositories in the
  `GenCr-ft` organization. (Completed for `../../gencraft-iac/.github/workflows/terraform-ci-cd.yml`).
- All reusable GitHub Actions stored within `gencraft-devops-automation` or other shared action
  repositories. (To be investigated if applicable).
- All automation scripts (Bash, Python, PowerShell, etc.) stored in `gencraft-devops-automation`. (Key
  Bash and PowerShell scripts audited in this version).
- Any other scripts or automated processes known to interact directly with GitHub repository names,
  structures, or APIs. (No cron jobs identified).
- Secrets, particularly GitHub Actions secrets, and their repository/organization scopes.
- Permissions used by automation tools or workflows, especially those tied to specific repository access.

## 3. Audit Checklist Structure

For each automation component identified, the following information MUST be collected and documented.
