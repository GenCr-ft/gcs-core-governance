# API Parameter Contracts --- Pending Authorship

This directory is reserved for per-service API parameter contract files (YAML/JSON).
These files are referenced by `agent-context/rules-index.md` under the `config-engines/api-parameters/` section.

## Current State

No parameter contract files have been authored yet.
This README exists to satisfy the governance index pointer and to prevent agents from
interpreting the empty directory as a missing or broken index entry.

## Planned Contents

Each file will document the parameter contracts for a specific service, including:
- Parameter names, types, and allowed values
- Required vs. optional classification
- Validation constraints
